## 다운로드

1. [firstcrackiscoming.com/download](https://firstcrackiscoming.com/download) 에 접속합니다.
2. **Download (.msi)** — `First-Crack-Windows.msi` 를 받습니다.

1. `.msi` 설치 프로그램을 실행하고 안내에 따라 설치를 완료합니다.
2. 시작 메뉴에서 **First Crack**을 실행합니다.
