## 다운로드

1. [firstcrackiscoming.com/download](https://firstcrackiscoming.com/download) 에 접속합니다.
2. Mac 칩에 맞는 설치 파일을 선택합니다.
   - **Apple Silicon** — `First-Crack-arm64.dmg`
   - **Intel Mac** — `First-Crack-x64.dmg`

macOS 12.0(Monterey) 이상이 필요합니다.

## 설치

1. `.dmg` 파일을 열고 First Crack 앱을 **Applications** 폴더로 드래그합니다.
2. 처음 실행 시 “확인되지 않은 개발자” 경고가 나오면 **시스템 설정 → 개인 정보 보호 및 보안**에서 “그래도 열기”를 허용합니다.
3. 앱을 실행하고 First Crack 웹 계정으로 로그인합니다.

## 로스팅 머신 연결

설치 후 **설정 → 로스팅 머신**에서 머신을 추가합니다. 연동 방법은 [로스팅 머신 연동](/machines/) 문서를 참고하세요.

Phidget를 사용하는 경우 macOS용 [Phidget22 드라이버](https://www.phidgets.com/downloads/phidget22/libraries/macos/Phidget22.dmg)를 먼저 설치해야 합니다.

## 업데이트

앱 내 업데이트 알림이 있으면 안내에 따라 설치합니다. 수동 설치가 필요하면 다운로드 페이지에서 최신 `.dmg`를 다시 받아 Applications 폴더의 기존 앱을 교체합니다.
