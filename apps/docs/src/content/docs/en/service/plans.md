Plans are based on the **desktop app + cloud sync**. Prices on the [landing page](https://firstcrackiscoming.com/#pricing) follow region and currency.

## Comparison

| | **Free (after trial)** | **Pro** | **Pro+** | **Enterprise** |
|---|------------------------|---------|----------|----------------|
| **Batch limit** | 20 / month | 200 / billing month | Unlimited | Unlimited |
| **Desktop app** | ✓ | ✓ | ✓ | ✓ |
| **Live roast & records** | ✓ | ✓ | ✓ | ✓ |
| **Profiles, greens, schedule** | Basic | ✓ | ✓ | ✓ |
| **Shopify & commerce** | — | — | ✓ | ✓ |
| **Seats** | 1 | 1 | 1 | Team |

:::note[Trial]
Checkout for paid plans may include a **14-day / 20-batch** trial. [Sign up](https://firstcrackiscoming.com/signup), then choose Pro or Pro+.
:::

## Pro

Core roast-floor tools — live roast, batch history, profiles, greens/blends, daily production plan. 200 batches/month, 1 seat.

## Pro+

Everything in Pro plus **unlimited batches** and **order/commerce connectors** (Shopify, Naver, Cafe24, etc.). For high volume or store sync needs.

## Enterprise

Multi-seat, custom features, on-site support (FDE), contract/invoice billing. Contact — [contact@firstcrackiscoming.com](mailto:contact@firstcrackiscoming.com)

## Free tier

After trial you can keep **20 batches/month** free. Upgrade to Pro when you need cloud limits and advanced capacity.
