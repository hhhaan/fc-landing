---

## Download

1. Open [firstcrackiscoming.com/download](https://firstcrackiscoming.com/download).
2. Pick the installer for your Mac chip.
   - **Apple Silicon** — `First-Crack-arm64.dmg`
   - **Intel Mac** — `First-Crack-x64.dmg`

Requires macOS 12.0 (Monterey) or later.

## Install

1. Open the `.dmg` and drag First Crack into **Applications**.
2. On first launch, if you see an “unidentified developer” warning, allow **Open Anyway** under **System Settings → Privacy & Security**.
3. Launch the app and sign in with your First Crack web account.

## Connect a roaster

After install, add a machine under **Settings → Roasting machines**. See [Roaster connection](/en/machines/).

For Phidget, install the [Phidget22 macOS driver](https://www.phidgets.com/downloads/phidget22/libraries/macos/Phidget22.dmg) first.

## Updates

Follow in-app update prompts. For a manual update, download the latest `.dmg` and replace the app in Applications.
