## 기본 원칙

### 원칙: 환불 없음

**구독료는 원칙적으로 환불되지 않습니다.**

이미 시작된 결제 기간의 **일할·부분 환불은 제공하지 않습니다.** 연간 선불도 동일합니다.

구독을 중간에 취소해도 **현재 결제 주기 종료일까지** Pro 기능을 계속 사용할 수 있으며, 남은 일수에 대한 금액은 환불되지 않습니다.

예: 월간 Pro를 15일째 취소 → 해당 기간 종료일까지 사용 가능, 차액 환불 없음.

### 체험

유료 플랜은 체크아웃 시 **14일 체험**이 포함될 수 있습니다. 체험 종료 전에 취소하면 첫 유료 청구를 막을 수 있습니다. 체험이 끝난 뒤 정상 청구된 금액은 환불 대상이 아닙니다.

## 예외 (재량 · 매우 한정)

아래 **둘 중 하나**에 해당할 때만 [support@firstcrackiscoming.com](mailto:support@firstcrackiscoming.com) 으로 요청할 수 있습니다. 단순 변심·기능 불만·미사용만으로는 **거절**합니다.

| 상황 | 처리 |
|------|------|
| **이중 청구** 또는 결제 시스템 오류로 인한 명백한 중복 결제 | 확인 후 **초과분**만 환불 검토 |
| 적용 법령이 청약 철회·환불을 **강제**하는 경우 | 법령이 정한 범위 내에서만 |

요청 시 **가입 이메일·결제일·플랜·사유·증빙**을 포함하세요. 처리 여부와 범위는 당사 재량이며, 약관·Polar 정책·법령이 우선합니다.

## 환불되지 않는 경우 (예시)

- 결제 기간 중 **일부라도** 서비스를 사용한 뒤의 단순 변심
- 체험 종료 후 **자동 갱신**된 정상 정기 결제
- 연간 결제 후 중도 해지·미사용 기간
- “기대와 다름”, 기기/네트워크 환경, 연동 설정 실수
- Enterprise **계약서**에 별도 조항이 있는 경우 — **계약 조건 우선**

## 환불 절차 (예외 해당 시)

1. 이메일: [support@firstcrackiscoming.com](mailto:support@firstcrackiscoming.com)
2. 포함 정보: 가입 이메일, 결제일, 플랜(월간/연간), 사유, 관련 영수증·스크린샷
3. 확인 후 Polar를 통해 안내
4. 환불이 승인되면 구독·접근은 즉시 또는 안내에 따라 종료될 수 있습니다

## 구독 취소 vs 환불

| | 구독 취소 | 환불 |
|---|----------|------|
| **의미** | 다음 갱신 중단, 현재 기간까지 사용 | 이미 낸 금액 반환 (원칙 불가) |
| **신청** | Polar 포털 또는 이메일 | 위 예외에 한해 이메일 |
| **남은 기간** | 사용 가능 (환불 없음) | 승인 시 접근 종료될 수 있음 |
