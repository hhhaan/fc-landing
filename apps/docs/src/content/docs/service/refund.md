First Crack 환불 정책은 [이용약관](https://firstcrackiscoming.com/terms)과 아래 내용을 따릅니다.

## 기본 원칙

### 부분 기간 환불 없음

**이미 시작된 결제 기간의 일할 환불은 제공하지 않습니다.**

구독을 중간에 취소해도 **현재 결제 주기 종료일까지** Pro 기능을 계속 사용할 수 있으며, 남은 일수에 대한 금액은 환불되지 않습니다.

예: 월간 Pro를 15일째 취소 → 해당 월 말일까지 사용 가능, 차액 환불 없음.

### 체험 기간

신규 가입 **14일·20배치 체험**은 신용카드 없이 시작할 수 있습니다. 체험 중 카드를 등록하지 않았다면 체험 종료 후 자동 결제되지 않습니다.

## 전액 환불 요청 (예외)

아래에 해당하면 [support@firstcrackiscoming.com](mailto:support@firstcrackiscoming.com) 으로 **결제일·계정 이메일·사유**를 보내 전액 환불을 요청할 수 있습니다. 개별 심사 후 처리합니다.

| 상황 | 권장 요청 기한 |
|------|----------------|
| **첫 유료 결제** 후 서비스가 기대와 다르거나 중복 결제 | 월간: 결제일 기준 **7일 이내** / 연간: **14일 이내** |
| **미사용** 첫 연간 결제 (배치 기록·Pro 기능 미사용) | 결제일 기준 14일 이내 |
| **이중 청구**·시스템 오류로 인한 중복 결제 | 발견 즉시 |

위 기한은 가이드라인이며, 약관·결제대행사(Polar) 정책이 우선합니다.

## 환불이 되지 않는 경우

- 결제 기간 중 **일부라도 Pro 기능을 사용**한 후 “단순 변심”만으로 남은 기간 환불 요청
- 체험 종료 후 **자동 갱신**된 정기 결제 — 취소는 가능하나 과거 기간 소급 환불 불가
- Enterprise **계약서**에 별도 조항이 있는 경우 — 계약 조건 우선

## 환불 절차

1. 이메일: [support@firstcrackiscoming.com](mailto:support@firstcrackiscoming.com)
2. 포함 정보: 가입 이메일, 결제일, 플랜(월간/연간), 요청 사유
3. 확인 후 Polar를 통해 카드 환불 또는 계좌 환급 안내
4. 환불 처리 후 구독은 즉시 또는 환불 정책에 따라 종료될 수 있습니다.

## 구독 취소 vs 환불

| | 구독 취소 | 환불 |
|---|----------|------|
| **의미** | 다음 갱신 중단, 현재 기간까지 사용 | 이미 낸 금액 일부·전액 반환 |
| **신청** | Polar 포털 또는 이메일 | 이메일로 환불 요청 |
| **남은 기간** | 사용 가능 (환불 없음) | 환불 시 접근 권한 종료될 수 있음 |
