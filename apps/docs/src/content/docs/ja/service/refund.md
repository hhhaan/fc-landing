返金は [利用規約](https://firstcrackiscoming.com/terms) と以下に従います。本ページと規約が異なる場合は **利用規約が優先** します。

## 基本方針

### 原則: 返金なし

**サブスクリプション料金は原則として返金しません。**

すでに開始した請求期間の **日割り・部分返金は行いません**（年間の前払いも含む）。

途中解約しても **現在の請求サイクル終了日まで** Pro を使え、未使用日数分は返金されません。

例: 月間 Pro を 15 日目に解約 → 期間終了まで利用可、差額返金なし。

### トライアル

有料プランはチェックアウト時に **14 日トライアル** が付く場合があります。トライアル終了前に解約すれば初回の有料請求を止められます。トライアル終了後に正常請求された金額は返金対象外です。

## 例外（裁量・極めて限定）

次の **いずれか** に該当する場合のみ [support@firstcrackiscoming.com](mailto:support@firstcrackiscoming.com) へ依頼できます。単純な気が変わり・機能不満・未使用のみでは **拒否** します。

| 状況 | 対応 |
|------|------|
| **二重請求** または決済システム起因の明白な重複課金 | 確認後 **超過分のみ** 返金を検討 |
| 適用法令がクーリングオフ等を **強制** する場合 | 法令が定める範囲内のみ |

**登録メール・決済日・プラン・理由・証跡** を含めてください。可否と範囲は当社裁量で、規約・Polar 方針・法令が優先します。

## 返金されない場合（例）

- 期間中に **少しでも** 利用した後の単純な気が変わり
- トライアル後の **自動更新** による正常な定期課金
- 年間決済後の途中解約・未使用期間
- 「期待と違う」、端末/ネットワーク、連携設定ミス
- Enterprise **契約書** に別条項がある場合 — **契約が優先**

## 手続き（例外該当時のみ）

1. メール: [support@firstcrackiscoming.com](mailto:support@firstcrackiscoming.com)
2. 含める情報: 登録メール、決済日、プラン（月間/年間）、理由、領収書・スクリーンショット
3. 確認後、Polar 経由で案内
4. 返金が承認された場合、アクセスは即時または案内に従い終了することがあります

## 解約 vs 返金

| | 解約 | 返金 |
|---|------|------|
| **意味** | 次回更新停止、現期間まで利用 | 既払い額の返還（原則不可） |
| **申請** | Polar ポータルまたはメール | 上記例外のみメール |
| **残り期間** | 利用可（返金なし） | 承認時に権限終了の場合あり |
