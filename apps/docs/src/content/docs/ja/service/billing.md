有料プラン（Pro / Pro+）は **Polar** 経由で決済されます。アカウントは Supabase 認証、サブスク状態は First Crack Web とデスクトップに反映されます。

## サブスク開始

1. [firstcrackiscoming.com](https://firstcrackiscoming.com) にログイン
2. [料金プラン](/ja/service/plans/) で Pro または Pro+ を選択
3. **Start trial** またはアップグレード → `/start-pro?plan=pro-monthly` / `pro-yearly`（または `pro-plus-monthly` / `pro-plus-yearly`）
4. **Polar チェックアウト** を完了（14 日トライアルが付く場合があります）
5. 成功後 [アカウントページ](https://firstcrackiscoming.com/account) で Pro 状態と次回請求日を確認

トライアル終了後、選択したプランの有料課金が自動で始まります。終了前に解約すれば初回の有料請求を止められます。

## 請求周期

| オプション | 説明 |
|------------|------|
| **月間** | 毎月更新、いつでも解約可 |
| **年間** | 1 年先払い、月換算で約 20% オフ |

Enterprise は月間・年間契約 — [営業](mailto:contact@firstcrackiscoming.com)

## 解約

1. [アカウントページ](https://firstcrackiscoming.com/account) で状態を確認
2. 解約・支払い方法変更は Polar カスタマーポータルまたは [support@firstcrackiscoming.com](mailto:support@firstcrackiscoming.com)

**解約時:** 現在の請求期間終了まで Pro 機能を利用できます。終了後は無料ティア（月 20 バッチ）になります。

:::caution[原則返金なし]
期間の途中で解約しても **日割り・部分返金はありません**。料金は原則返金不可です。詳細は [返金ポリシー](/ja/service/refund/) を参照してください。
:::

## 支払い方法の変更

Polar ポータルでカード変更ができます。リンクが必要なら [support@firstcrackiscoming.com](mailto:support@firstcrackiscoming.com) まで。

## 支払い失敗（past_due）

限度超過・期限切れ・拒否などで失敗すると:

1. アカウントに **Payment issue** バナーが出ることがあります
2. 有効な支払い方法に更新するかサポートへ連絡
3. 長期未払いではサブスクが解約され無料ティアになります

## 領収書・請求書

Polar チェックアウト・ポータルで領収書を確認できます。韓国事業者向けの税金計算書が必要な場合は、事業者登録番号を添えて [support@firstcrackiscoming.com](mailto:support@firstcrackiscoming.com) へ。

## Enterprise・請求書払い

契約・銀行振込・インボイスは Enterprise 専用です。[contact@firstcrackiscoming.com](mailto:contact@firstcrackiscoming.com)
