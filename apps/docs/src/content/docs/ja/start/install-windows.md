## ダウンロード

1. [firstcrackiscoming.com/download](https://firstcrackiscoming.com/download) を開きます。
2. **Download (.msi)** — `First-Crack-Windows.msi` を取得します。

Windows 10/11、64-bit x86 をサポートします。

## インストール

1. `.msi` を実行し、ウィザードに従って完了します。
2. スタートメニューから **First Crack** を起動します。
3. First Crack Web アカウントでログインします。

## 焙煎機の接続

インストール後、**設定 → 焙煎機** でマシンを追加します。USB シリアルのロースターは [シリアルポートドライバー](/ja/machines/serial-driver/) のインストールが先に必要な場合があります。

## アップデート

アプリ内の更新通知に従ってください。手動の場合は既存インストール上で最新 `.msi` を再実行します。

## Windows ファイアウォール

Modbus TCP 接続時、Windows ファイアウォールが First Crack のネットワークアクセスを拒否することがあります。接続できない場合は例外に追加してください。
