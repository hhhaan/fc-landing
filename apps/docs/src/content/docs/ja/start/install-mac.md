## ダウンロード

1. [firstcrackiscoming.com/download](https://firstcrackiscoming.com/download) を開きます。
2. Mac のチップに合うインストーラを選びます。
   - **Apple Silicon** — `First-Crack-arm64.dmg`
   - **Intel Mac** — `First-Crack-x64.dmg`

macOS 12.0 (Monterey) 以上が必要です。

## インストール

1. `.dmg` を開き、First Crack を **Applications** にドラッグします。
2. 初回起動で「確認されていない開発者」と出た場合は、**システム設定 → プライバシーとセキュリティ** で「このまま開く」を許可します。
3. アプリを起動し、First Crack Web アカウントでログインします。

## 焙煎機の接続

インストール後、**設定 → 焙煎機** でマシンを追加します。手順は [焙煎機の接続](/ja/machines/) を参照してください。

Phidget を使う場合は、先に [Phidget22 macOS ドライバ](https://www.phidgets.com/downloads/phidget22/libraries/macos/Phidget22.dmg) をインストールしてください。

## アップデート

アプリ内の更新通知に従ってください。手動の場合は最新の `.dmg` をダウンロードし、Applications 内のアプリを置き換えます。
