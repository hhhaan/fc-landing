## 焙煎機接続の相談

接続が難しいときは次を添えてください。

- ロースター **メーカー・モデル**
- **OS**（Windows 11 / macOS 14 など）と First Crack **アプリバージョン**
- 接続方式: USB / LAN / Phidget
- **Artisan Machine** 設定のスクリーンショット（使用中の場合）→ [Artisan ガイド](/ja/machines/artisan/)
- エラーメッセージ・ポート一覧のスクショ

## ログの送付

クラッシュやエラー時、サポートがログを依頼することがあります。

1. アプリ **設定 → 詳細 → ログ送信**（または案内されたパス）
2. 生成ファイルをメールに添付
3. 発生 **時刻・再現手順** を記載
