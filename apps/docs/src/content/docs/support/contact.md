## 로스팅 머신 연동 문의

연동이 어려울 때 아래를 함께 보내 주시면 빠르게 도와드릴 수 있습니다.

- 로스터 **제조사·모델**
- **OS** (Windows 11 / macOS 14 등) 및 First Crack **앱 버전**
- 연결 방식: USB / LAN / Phidget
- **Artisan Machine 설정** 스크린샷 (사용 중인 경우) → [Artisan 가이드](/machines/artisan/)
- 오류 메시지·포트 목록 스크린샷

## 로그 파일 전달

앱 오류·충돌 시 지원팀이 로그를 요청할 수 있습니다.

1. 앱 **설정 → 고급 → 로그보내기** (또는 지원팀 안내 경로)
2. 생성된 파일을 이메일에 첨부
3. 문제 발생 **시각·재현 단계**를 함께 기재
